EcoLearn - Web (S.Id style)
============================
Isi paket:
- index.html
- materi.html
- kuis.html
- tips.html
- tentang.html

Instruksi cepat:
1. Buka index.html di browser untuk melihat demo lokal.
2. Upload semua file ke GitHub Pages atau Netlify untuk publikasi.
3. Buat short link di s.id (https://s.id) setelah alamat publik aktif.

Rekomendasi:
- Untuk mempercantik, tambahkan folder assets/ dengan logo dan gambar.
- Untuk versi mobile app, gunakan tools seperti PWA or WebView wrapper.
